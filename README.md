## Made For

A scanner to find subdomains of a site. This software is able to identify more than 220,000 subdomains.


**Requires**
> Visual Studio 2019 - Telerik WinForm - Chilkat - .NetFramework 4.7 or above
>


Features

- Support over 220,000 subdomain
